# streaming-sensors-hub

this will collect data from an arduino and/or a neurosky mindwave

## quickstart

[download this binary and run](http://people.ischool.berkeley.edu/~nick/sensors-hub.zip) (OSX).

Right click and press "open."

## developing

clone this repo and 

```
npm install
```

to run

```
npm start
```
